# Lumina Clippers - Marketing Site

## Overview
Marketing website for Lumina Clippers content distribution network.

## Tech Stack
- Next.js 16
- React 19
- Tailwind CSS
- Deployed on Vercel

## Local Development
```bash
npm install
npm run dev
```

## Environment Variables
Copy `env.example` to `.env.local` and fill in values.

## SEO
- sitemap.xml and robots.txt in /public
- OG tags configured in app/layout.tsx

## Pages
- / - Home
- /about - About Us
- /casestudies - Case Studies
- /contact - Contact
